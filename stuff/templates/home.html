pwnworld is a platform for learning Information Security through some CTF style challenges. Each challenge contains some security vulnerability which has to be exploited by the player in order to get the solution. A flag (a string which indicates that the user has successfully solved a challenge) is provided to the user if he successfully solves the challenge.
