pwnworld is a platform for learning Information Security through some CTF style challenges. Each challenge contains some security vulnerability which has to be exploited by the player in order to get the solution. A flag (a string which indicates that the user has successfully solved a challenge) is provided to the user if he successfully solves the challenge.

This awesome platform has been designed and developed by four students of Indian Institute of Technology, Roorkee.

    • RuLes

  • Do not be rude towards other players.
  • Do not use unethical language in the chat section.
  • Do not attack the infrastructure or any challenge by using automatic scanners such as Nmap, gobuster, dirbuster, sqlmap, nosqlmap, brute-forcers etc.
  • DDOS, DOS attacks are strictly prohibited.
  • Some specific challenges would require the use of these tools. It would be clearly mentioned in the challenge if it is required.
  • Attacking the platform infrastructure may lead to temporary or permanent ban.
  • In case you are temporarily banned, contact the admins via discord and provide the assurance that you'll not do any such activity again, which may lead to a ban
  • Brute forcing flags is prohibited.
  • Please do not post any public writeups for any challenge.
  • Kindly agree to the above rules, and enjoy learning Information Security.
  • You may create your own challenges and submit them. The challenges will be deployed after verification and malware analysis(in case of pwn and RE).
